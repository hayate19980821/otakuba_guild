Update 5.6 初回セットアップウィザード

追加内容:
- 初めて店舗側が開いた時に、初回セットアップ画面を表示
- 設定できる項目:
  1. 店舗名 / テーマ
  2. スタート画面タイトル / メッセージ / 背景 / BGM
  3. 「いいえ」を押した時の表示名 / 画像 / セリフ
  4. 討伐完了画面の背景 / 中央画像 / タイトル / メッセージ / BGM
  5. 店舗ID / GAS URL / Discord URL
- 保存すると settings.themeCustom と店舗設定に反映
- GAS同期で全端末に反映
- 管理画面 → Game設定 から後から編集可能
- 管理画面 → 設定 から初回セットアップを再表示可能

アップロードするファイル:
- index.html
- app.js
- storage.js
- admin.js
