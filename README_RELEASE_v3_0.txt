おたく場ギルド v3.0 RELEASE候補

使い方
1. このZIPの中身をGitHubリポジトリ直下へ全部上書き
2. GitHub Pagesで index.html?v=3.0 を開く
3. 音が鳴らない時は、最初に画面を一度タップしてください（iPhoneの自動再生制限対策）

入っている主な機能
- 注文追加 / 受注一覧 / 注文確認 / 会計
- 会計金額による敵HPダメージ
- 撃破で次ステージへ進行
- monsters.jsonベースの背景・敵画像・BGM設定
- タイトルBGM / 敵別BGM / 効果音 / レベルアップSE / 魔王撃破ファンファーレ / エンディングBGM
- 管理画面 admin.html

注意
- iPhoneではBGM/SEは最初のタップ後に再生可能になります。
- 画像や音源ファイル名は大文字小文字・日本語表記も一致させてください。
- GAS連携は README_GAS_SETUP.txt を見てください。
